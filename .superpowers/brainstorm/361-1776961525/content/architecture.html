System Architecture

Two automatic trigger paths, one shared voice engine underneath.

Claude Code Path
stop hook fires

Claude Code calls the hook script with the full response text when Claude finishes generating

↓
Text filter

Strip code fences, inline code, tool call blocks, file diffs — keep only prose sentences

↓ full filtered response
ElevenLabs → audio

Calls API with selected voice, streams MP3, plays immediately

Claude Desktop Path
UIAutomation daemon

Python background process watches Claude Desktop's accessibility tree for new streamed text

↓ sentence completes
Text filter (same)

Same filter logic — rejects sentences inside code/tool blocks, speaks clean prose sentences as they complete

↓ sentence by sentence
ElevenLabs → audio

Each sentence queued and spoken as it arrives — feels like real-time conversation

Shared Voice Engine (Python module)

filter.py
Strips code blocks, tool calls, diffs. Returns clean prose only.
tts.py
ElevenLabs API call + MP3 playback. Audio queue for sentence streaming.
config.json
Stores: enabled flag, selected voice ID, ElevenLabs API key.
