Where does voice need to work?

Select all the Claude surfaces you want voice support on. This shapes the whole architecture.

A

Claude Code CLI

Terminal-based. Has hooks — shell scripts that fire when Claude stops responding. Most powerful integration point.

B

Claude Desktop App

Windows/Mac native app. No official hooks, but screen/clipboard monitoring is possible.

C

Claude.ai Web

Browser-based. Would require a browser extension or separate tool to intercept responses.

Note

Claude Code hooks are the cleanest integration — they fire automatically when Claude finishes a response and can capture the full text. Other surfaces require workarounds.
